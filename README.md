## Interactive Flashcard Platform for Learning 
 Proyecto Final de Programación Web

Plataforma de aprendizaje con tarjetas interactivas desarrollada para facilitar el estudio mediante la creación, gestión y práctica de flashcards digitales.
El sistema permite a los usuarios registrarse, autenticar su sesión y crear tarjetas personalizadas con preguntas, respuestas, categorías y niveles de dificultad.

Cada tarjeta puede ser marcada como favorita o revisada más tarde, y el usuario puede realizar sesiones de estudio interactivas que registran su rendimiento.
El objetivo principal es ofrecer una herramienta flexible y escalable que combine almacenamiento estructurado de datos, interactividad y seguimiento del progreso del aprendizaje.

##  Características principales

 * Gestión de usuarios: registro, inicio de sesión y autenticación segura.

 * Creación y edición de tarjetas: cada tarjeta contiene una pregunta, respuesta, categoría y nivel de dificultad.

 * Organización por categorías: permite clasificar las tarjetas según temas o materias.

 * Sistema de favoritos: los usuarios pueden marcar tarjetas importantes para revisarlas más tarde.

 * Sesiones de práctica interactivas: el usuario puede estudiar sus tarjetas de forma dinámica, con retroalimentación inmediata.

 * Seguimiento del progreso: registro del rendimiento y estadísticas básicas del aprendizaje.

 * Almacenamiento estructurado: base de datos relacional para manejar usuarios, tarjetas, categorías y dificultad.

 * Arquitectura modular: separación clara entre modelos, vistas y lógica de negocio.

 * Diseño escalable: preparado para integrar nuevas funcionalidades (como repaso espaciado, ranking o gamificación).


## Tecnologías utilizadas

* Lenguaje principal: Python 

* Framework web: Django REST Framework 

* Base de datos: SQLite 

* Autenticación: sistema integrado de Django

* API REST: endpoints para la gestión de usuarios, tarjetas, categorías y estadísticas

* Control de versiones: Git + GitHub

* Dependencias gestionadas con: pip y archivo requirements.txt
